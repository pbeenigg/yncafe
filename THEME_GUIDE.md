# 云南咖啡品牌档案库 - 主题设计指南

## 🎨 主题概览

本项目提供了6种精心设计的主题，其中3种是根据云南咖啡豆种特性创新设计的专属主题：

### 基础主题
- **浅色模式** - 经典的温暖琥珀色调
- **深色模式** - 深色背景配合琥珀色点缀  
- **系统模式** - 自动跟随系统主题设置

### 咖啡豆种主题 ☕

#### 🌸 瑰夏主题 (Geisha Theme)
**设计理念**：瑰夏咖啡以其独特的花香果香闻名，被誉为"咖啡界的香奈儿"
- **主色调**：优雅的粉紫色系
- **灵感来源**：花香、果香、优雅气质
- **色彩特点**：
  - 背景：淡粉紫渐变 `oklch(0.97 0.02 320)`
  - 主色：瑰夏花香色 `oklch(0.55 0.15 325)`
  - 辅助色：浅花香色、果香色
- **适用场景**：追求优雅、精致体验的用户

#### 🍃 铁皮卡主题 (Typica Theme)  
**设计理念**：铁皮卡以明亮的柑橘酸度和清新花香著称，是精品咖啡的代表
- **主色调**：清新的绿橙色系
- **灵感来源**：柑橘酸度、花香、清新感
- **色彩特点**：
  - 背景：淡柑橘绿渐变 `oklch(0.96 0.03 85)`
  - 主色：柑橘色 `oklch(0.50 0.18 95)`
  - 辅助色：明亮柑橘、浅柑橘色
- **适用场景**：喜欢清新、明亮视觉体验的用户

#### 🍇 波旁主题 (Bourbon Theme)
**设计理念**：波旁咖啡甜度高、醇厚度好，带有巧克力香气，口感平衡
- **主色调**：温暖的紫棕色系
- **灵感来源**：巧克力香气、甜美、醇厚感
- **色彩特点**：
  - 背景：淡紫棕渐变 `oklch(0.95 0.03 280)`
  - 主色：巧克力紫色 `oklch(0.48 0.14 285)`
  - 辅助色：浅巧克力色、甜美紫色
- **适用场景**：偏爱温暖、甜美色调的用户

## 🎯 设计原则

### 色彩心理学应用
1. **瑰夏** - 粉紫色传达优雅、高贵、精致的感觉
2. **铁皮卡** - 绿橙色营造清新、活力、自然的氛围
3. **波旁** - 紫棕色体现温暖、舒适、甜美的特质

### 可访问性考虑
- 所有主题都确保足够的对比度，符合 WCAG 2.1 AA 标准
- 色彩搭配考虑色盲用户的使用体验
- 提供清晰的视觉层次和信息架构

### 品牌一致性
- 保持咖啡文化的核心元素
- 每个主题都体现对应豆种的特色
- 统一的交互模式和视觉语言

## 🛠 技术实现

### CSS 变量系统
使用 OKLCH 色彩空间定义主题色彩，提供更好的色彩一致性和感知均匀性。

### 主题切换机制
- 基于 `next-themes` 实现无缝主题切换
- 支持系统主题自动检测
- 主题状态持久化存储

### 响应式适配
- 所有主题都完全支持响应式设计
- 在不同设备和屏幕尺寸下保持最佳视觉效果

## 🎨 使用建议

- **瑰夏主题**：适合展示高端精品咖啡内容
- **铁皮卡主题**：适合日常浏览和学习咖啡知识
- **波旁主题**：适合温馨的咖啡文化体验

每个主题都经过精心调试，确保在不同光线环境下都能提供舒适的阅读体验。
