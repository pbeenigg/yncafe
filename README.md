### **品种风味特点（基于专业资料和最新市场调研）：**

### 铁皮卡 (Typica)
- **风味描述**：明亮的柑橘酸度，优雅的花香，甜感突出，口感干净
- **价格区间**：生豆150-260元/公斤，熟豆75-130元/500g
- **市场地位**：高端精品咖啡代表，深受专业咖啡师喜爱

### 波旁 (Bourbon)
- **风味描述**：甜度高，醇厚度好，风味平衡，酸度柔和，带有巧克力香气
- **价格区间**：生豆120-200元/公斤，熟豆60-100元/500g  
- **市场地位**：传统优质品种，适合日常饮用和精品咖啡

### 卡蒂姆 (Catimor)
- **风味描述**：口感醇厚，苦味适中，性价比优秀，适合拼配
- **价格区间**：生豆60-90元/公斤，熟豆30-90元/500g
- **市场地位**：云南主流品种，产量大，适合大众市场

### 瑰夏 (Geisha)
- **风味描述**：花香果香明显，层次丰富，价格较高，风味独特
- **价格区间**：生豆300-800元/公斤，熟豆150-400元/500g
- **市场地位**：顶级精品咖啡，稀有品种，收藏价值高

### 罗布斯塔 (Robusta)
- **风味描述**：咖啡因含量较高，苦感与醇厚度更强，常见坚果、可可、木质与香料调；在云南西双版纳等地经精细处理（精品罗布斯塔）可呈现黑糖、热带水果与草本风味，干净度明显提升。
- **价格区间**：生豆30-80元/公斤，熟豆25-90元/500g（精品等级略高）
- **市场地位**：广泛用于商业拼配与意式基底；云南产区正推动精品化，逐步进入精品市场与国际买家视野。

### 萨奇姆 (Sarchimor)
- **风味描述**：来自蒂莫尔杂交系与维拉萨奇的后代，兼具抗病性与较佳杯质；常见柑橘、坚果、红茶与可可调性，酸质中等、体质饱满，适合中度至中深度烘焙。
- **价格区间**：生豆70-120元/公斤，熟豆40-110元/500g
- **市场地位**：云南新近推广的适栽品系之一，产量稳定、病害抗性好，适合做意式与手冲的全能型豆。

### 铁皮卡古树 (Typica Old Trees)
- **风味描述**：在云南高龄咖啡树上采收的铁皮卡，甜感与层次更佳，常见花香、蜂蜜、柑橘、香草与优雅香料，余韵绵长，洁净度高。
- **价格区间**：生豆200-400元/公斤，熟豆120-300元/500g（视树龄与批次而定）
- **市场地位**：稀缺高端品项，多用于限量款、礼盒与比赛批次，面向精品与收藏市场。

**处理法说明：**
- 水洗：水洗处理，口感干净明亮
- 日晒：日晒处理，果香浓郁
- 蜜处理：蜜处理，甜度较高
- 橡木桶发酵：特殊发酵工艺，风味独特

**主要产地分布：**
- 保山：铁皮卡、波旁等品种
- 普洱：卡蒂姆为主，产量最大
- 临沧：波旁、卡蒂姆混合种植
- 德宏：卡蒂姆为主，速溶咖啡原料
- 南屏：铁皮卡、波旁等品种
- 西双版纳：罗布斯塔、卡蒂姆等品种
